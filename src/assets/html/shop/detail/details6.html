Media Pod

B tv, wavve를 통해 다시 보고 싶은 드라마의 명대사와 명장면을 만나보실 수 있습니다. 명대사 카드 중 마음에 드는 것을 골라보세요. 바로 해당 장면을 확인하실 수 있습니다.

이용 방법

  1. 시대를 뛰어넘어 사랑받는 드라마, 영화 속 명대사가 쓰여 있는 카드를 둘러보고, 마음에 드는 카드를 골라주세요.
  2. 메인 화면의 START 버튼을 눌러주세요.
  3. 트레이에 카드 윗면의 QR코드가 잘 보이도록 올리면 B tv, wavve 콘텐츠로 이동합니다.
  4. 카드 아래쪽의 시간에 맞춰 재생 바를 옮기면 내가 고른 대사를 직접 들어볼 수 있습니다.
